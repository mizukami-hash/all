メモ帳アプリ

スキル

  • ローカルストレージ
  • JSON

作品URL

GitHub 　：https://github.com/mizukami-hash/memo.git


参考サイト：https://www.youtube.com/watch?v=Efo7nIUF2JY

制作期間

3日

学習内容

クリックでページが増えて、ダブルクリックで削除できるメモ帳アプリです。
ローカルストレージにデータを保存しているため、リロードしても消えない仕様になっています。
処理が複雑で最初は難しく感じましたが、DOMやローカルストレージの理解を深めることができました。
メモ／追加修正：ダブルクリックの処理をわかりやすいものに変更・最終更新日時の追加
